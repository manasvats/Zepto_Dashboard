# Zepto_Dashboard
This is a dashboard for Zepto which is a Sales performance Dashboard.
📊 Zepto Sales Performance Dashboard (Excel)
🏢 Overview

Interactive Excel dashboard analyzing sales & operations performance of Zepto.

Built using Pivot Tables, Charts, KPI Cards, and Slicers to generate business insights from raw transactional data.

📌 Final Conclusion

The dashboard reveals clear revenue concentration in specific states and cities, with Snacks and Beverages emerging as top-performing categories.

While overall revenue and order volume show strong performance, discount levels remain moderate (4.92%), indicating potential opportunities for targeted promotional strategies to boost underperforming categories.

Geographic insights suggest expansion and marketing efforts should prioritize high-demand states while optimizing operations in lower-performing regions.

Overall, this project demonstrates how Excel can be leveraged to convert raw data into strategic business decisions for quick-commerce growth.
